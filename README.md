# Jobly Solutions

A React + TypeScript web application for Jobly Solutions — staffing, consulting & software training.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Radix UI / shadcn-ui
- React Router
- React Hook Form + Zod

## Getting Started

```bash
npm install --legacy-peer-deps
npm run dev
```
