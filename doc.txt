Employee Onboarding

Objective

To onboard new employees or consultants into the system with all necessary details, documentation, and system access.

Key Data Fields

Employee Profile includes:

Personal Information

Full Name
Employee ID (Auto Generated)
Email Address
Phone Number
Address
Date of Birth
Employment Details

Employment Type (Full-time / Contract / Vendor)
Job Title
Department
Joining Date
Reporting Manager
Work Location
Immigration / Compliance (Optional)

Visa Status
Work Authorization Expiry
SSN / Last 4 digits
I-9 Documents
Payroll Information

Payment Type (W2 / 1099 / C2C)
Hourly Rate / Salary
Bank Details (ACH)
Tax Forms (W9 / W4)
Documents Upload

Resume
Offer Letter
ID Proof
Compliance Documents
Workflow

HR logs into the portal.
Navigate to Employee Management → Add New Employee.
Enter employee details.
Upload required documents.
System generates Employee ID.
Employee receives portal login credentials via email.
Employee logs in and completes profile.
HR approves onboarding.
Employee becomes Active in the system.

Employee Tagging to Client (Project Assignment)

Objective

To assign employees to clients/projects for time tracking and billing.

Assignment Fields

Employee Name
Client Name
Project Name
Role
Start Date
End Date
Bill Rate
Pay Rate
Billing Type
Work Location
Reporting Manager
Workflow

Operations Manager navigates to Assignments → Create Assignment.
Select Employee.
Select Client.
Enter project details.
Enter billing rate and pay rate.
Define assignment duration.
Save assignment.
System Actions

Employee is now tagged to the client.
Employee can submit timesheets against this project.
Billing will be generated based on this assignment.

Invoicing Module

Objective

To generate invoices for clients based on approved timesheets.

Invoice Fields

Invoice Number (Auto Generated)
Client Name
Billing Period
Employee Name
Hours Worked
Bill Rate
Total Amount
Taxes
Payment Terms
Invoice Date
Due Date
Workflow

Finance navigates to Billing → Generate Invoice.
Select Client.
System pulls approved timesheets.
System calculates invoice amount.
Calculation Example

Hours Worked × Bill Rate = Invoice Amount

Example
160 hours × $80/hr = $12,800

System generates invoice PDF.
Invoice sent to client via email.
Invoice status updated.
Status Types

Draft
Sent
Paid
Overdue
